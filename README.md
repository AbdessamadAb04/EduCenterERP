# EduCenterERP

EduCenterERP est une solution complète de gestion pour les établissements éducatifs, offrant des modules pour la gestion des étudiants, des classes, des finances, des emplois du temps et bien plus encore.

## Fonctionnalités principales
- Gestion des étudiants et des classes
- Suivi des paiements et des finances
- Génération de reçus et d’exports PDF
- Tableau de bord analytique
- Gestion des emplois du temps
- Interface d’administration
- Support utilisateur intégré

## Technologies utilisées
- React (frontend)
- TypeScript
- Vite
- Spring Boot (backend)
- Java
- Maven

## Démarrage rapide

### Prérequis
- Node.js et npm
- Java 17+
- Maven

### Installation

#### Frontend
```bash
cd educenter-frontend
npm install
npm run dev
```

#### Backend
```bash
cd educenter-backend
./mvnw spring-boot:run
```

## Structure du projet
- `educenter-frontend/` : Application web (React/TypeScript)
- `educenter-backend/` : API backend (Spring Boot/Java)
- `documentary/` : Documentation et notes
